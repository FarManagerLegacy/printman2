Плагин написан для печати документов и выделенной области
взамен плагина идущего в поставке с Far.
Плагин идущий в поставке некоректно отрабатывает при большом количестве
принтеров в системе.
Для успешной работы плагина необходимо удалить все файлы из каталога
PrintMan и поместить туда имеющиеся в текущем архиве документы.

Версия 1.0 21.08.2009
Плагин умеет печатать текущий файл из панели и выделенный текст из редактора.
Сохраняет последний выбранный принтер и ставит его по умолчанию при следующей
печати
